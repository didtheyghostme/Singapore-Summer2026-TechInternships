## Community rows (you CAN add / edit / remove)
A row is a **community row** if the **Track** column is `-`.

### Community row rules (required)
- **Company**: must be a markdown link: `[Company Name](https://company.com)`
- **Role**: free text
- **Track**: must be exactly `-`
- **Application**: must be a **plain** `http(s)` URL (no HTML/buttons)
- **Date Added**: must be `YYYY-MM-DD` (real date)
